# Phase 1 - Project Foundation

Phase 1 membangun fondasi monorepo ATIGA Assessment AI:

- Backend Fastify + TypeScript strict.
- Frontend Next.js App Router + Tailwind CSS.
- PostgreSQL 16 + pgvector sebagai satu-satunya backing service.
- Drizzle ORM untuk schema dan migration aplikasi.
- pg-boss untuk queue berbasis PostgreSQL tanpa Redis.
- Skill system awal berbasis file Markdown dengan YAML frontmatter.
